# Bax Portfolio

Minimal, Apple-inspired personal portfolio scaffolded with plain HTML, CSS, and light JavaScript. Designed to highlight intro, about, skills, projects, experience, and contact sections with a clean responsive layout.

## Getting Started

No build tools required. Open `index.html` in a browser or serve the folder with any static server.

```bash
# macOS example
python3 -m http.server 4173
```

## Customizing

- Replace placeholder copy, metrics, and project details in `index.html`.
- Adjust theme tokens and layout tweaks in `styles/main.css`.
- Extend interactions (nav state, filters, animations) via `scripts/main.js`.
- Add assets (portraits, logos, case study images) inside `assets/` and reference them in markup.
